# Adami-Canvas

Adami-Canvas 是一个本地运行的多模态无限画布工具，用于组织文本、图片、视频、音频素材，并将它们打包发送给火山方舟 / Seedance 2.0 等模型生成内容。

## 核心功能

- 无限画布节点编辑
- 图片、视频、音频素材库
- 文本节点内嵌 `@素材` 引用
- 音频上传节点
- 火山方舟 API 设置
- Seedance 2.0 视频生成节点

## 启动方式

普通用户请从 GitHub 的 **Releases** 页面下载 `Adami-Canvas-Windows-Full-*.zip`。完整版自带 Python 运行环境，解压后即可启动；仓库首页的“Code → Download ZIP”只有源码，不包含内置运行环境。

双击：

```text
START.bat
```

启动成功后浏览器会打开：

```text
http://127.0.0.1:3000/
```

使用时请保持启动窗口打开。

## 一键更新与数据保护

首页左侧的「一键更新」会检查本仓库的新版本，自动备份旧程序、安装更新并重启。更新器只替换程序代码，以下本地内容不会上传，也不会被更新覆盖：

- `data/`：画布与项目
- `assets/`：素材库
- `API/.env`：API Key 与本机配置
- `output/`、`logs/`：生成结果与日志
- `python/`、`packages/`：本机运行环境

## API 设置

1. 打开首页左侧的「API 设置」。
2. 填写火山方舟 API Key / Access Key 信息。
3. 拉取或验证模型列表。
4. 回到「Adami-Canvas」画布，添加文本、图片、音频和视频生成节点。

默认接口地址：

```text
https://ark.cn-beijing.volces.com/api/v3
```

## 多模态输入

文本节点支持输入 `@` 调用素材库。选择素材后，图片和音频会以内嵌标签的形式显示在文本流里，同时底层保留素材地址，生成节点会自动整理为模型请求所需的文本、图片、视频和音频参数。

## 常见提示

- 如果模型列表为空，请先到「API 设置」里拉取模型。
- 如果要做 Seedance 视频，优先使用「视频生成」节点。
- 如果要复用素材，先放进素材库，再通过 `@素材` 或拖拽方式加入画布。
- 输出结果会保存到本地输出目录。
